Кратко.
Модуль Flexbox Layout (гибкая коробка) направлен на эффективное размещение, выравнивание и распределение элементов в контейнере.
Основная идея гибкой компоновки - дать контейнеру возможность изменять ширину/высоту и порядок расположения элементов для наилучшего заполнения доступного пространства.
Flexbox не зависит от направления, но не подходит для сложных приложений из-за отсутствия гибкости.
Макет Flexbox подходит для компонентов приложения и мелкомасштабных макетов, в то время как макет сетки предназначен для крупномасштабных макетов.
Flexbox содержит множество свойств для родительского элемента (гибкий контейнер) и дочерних элементов (гибкие элементы).
Свойства flexbox включают display, flex-direction, flex-wrap, flex-flow, flex-grow, flex-shrink, flex-basis, align-items, justify-content, order, align-self, flex-direction и другие.
Поддержка браузеров для flexbox варьируется, но большинство браузеров поддерживают основные свойства.
Flexbox позволяет легко центрировать элементы, использовать выравнивание и гибкую адаптацию макетов.
